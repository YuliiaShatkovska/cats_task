## Project Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/cats_task.git
   cd cats_task
   ```
2. Install dependencies:  
  ```bash
    npm install
  ```
3. Start the development server:
  ```bash
    npm run dev
  ```

The app will be available at http://localhost:5173.
